Program zawiera 3 klasy - Ułamek, BigLiczba, Macież tesotwane po kolei w klasie Main.

### Ułamek:

- rozwinięcie dziesiętne
- dodawanie
- odejmowanie
- mnożenie
- skracanie
- odwracanie

### BigLiczba

- sprawdzenie podzielności
- sprawdzenie pierwszości

### Macierz 3x3

- wyznacznik
- transpozycja
- dodawanie
- odejmowanie
- mnożenie
